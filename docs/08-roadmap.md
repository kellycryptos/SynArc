# Project Roadmap

This section outlines SynArc's multi-phase roadmap towards providing secure funding, automated treasury protection, and seamless community coordination tools.

***

## Roadmap Milestones

### Phase 1: Frictionless Coordinator Onboarding

* **Status:** _Planned_
* **Description:** Remove wallet setup complexity, enabling creators and backers to join using social logins or email with automatic on-chain setup.

### Phase 2: Secure Milestone Escrows

* **Status:** _Planned_
* **Description:** Lock all community contributions in secure escrows that release funds in tranches only after community approval votes.

### Phase 3: Delegated Community Voting

* **Status:** _Planned_
* **Description:** Enable members to delegate voting weights directly to trusted coordinators programmatically without transferring their tokens.

### Phase 4: Cross-Chain Funding Campaigns

* **Status:** _Planned_
* **Description:** Accept community funding and support from other networks seamlessly, routing funds directly to the target escrow.

### Phase 5: One-Click Workspace Factory

* **Status:** _Planned_
* **Description:** Allow any group to deploy a pre-configured workspace with built-in escrows, token voting, and treasury logs in one click.

### Phase 6: Automated Treasury Guard API

* **Status:** _Planned_
* **Description:** Deploy developer SDK modules to easily create, connect, and customize automated rebalancing rules and guards for workspace treasuries.
